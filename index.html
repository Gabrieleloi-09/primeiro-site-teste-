Lista de Comandos do Visual Studio Code

    1. Ctrl + Shift + P: Abre a caixa de comando, onde é possível pesquisar por funcionalidades específicas.
    2. Ctrl + D: Seleciona a próxima ocorrência da palavra selecionada.
    3. Ctrl + Shift + D: Duplica a linha atual.
    4. Ctrl + Shift + L: Seleciona todas as ocorrências da palavra selecionada.
    5. Ctrl + Shift + O: Ordena as linhas selecionadas em ordem crescente.
    6. Ctrl + Shift + F: Abre a ferramenta de busca global, que permite pesquisar por um termo em todos os arquivos do projeto.
    7. Ctrl + Shift + X: Abre a barra de extensões, onde é possível instalar e gerenciar extensões do VS Code.
    8. Ctrl + N Abre um novo arquivo.
    9. Ctrl + Shift + N Abre uma nova janela do VS Code.
    10. Ctrl + S Salva o arquivo atual.
    11. Ctrl + Shift + S Salva todos os arquivos abertos.
    12. Ctrl + X Recorta a seleção atual.
    13. Ctrl + C Copia a seleção atual.
    14. Ctrl + V Cola a seleção atual.
    15. Ctrl + Z Desfaz a última ação.
    16. Ctrl + Shift + Z Refaz a última ação.
    17. Ctrl + F Abre a barra de pesquisa.
    18. Ctrl + Shift + F Abre a pesquisa em todos os arquivos.
    19. Ctrl + G Vai para uma linha específica do arquivo.
    20. Ctrl + Shift + L Seleciona todas as ocorrências da seleção atual.
    21. Ctrl + / Comenta ou descomenta a seleção atual.
    22. Alt + Shift + seta para cima Move a linha atual para cima.
    23. Alt + Shift + seta para baixo Move a linha atual para baixo
    24. F12 Vai para a definição do símbolo.
    25. Alt + F12 Abre a definição do símbolo em uma nova janela.
    26. Ctrl + Shift + O Abre a lista de símbolos do arquivo atual.
    27. Ctrl + 1 para acessar os arquivos duplicados
    28. Ctrl + ] joga o arquivo para o lado
    29. Ctrl +F4 OU Ctrl + W fecha a aba selecionado.
    30. Ctrl + Shift + T Reabre a aba fechada
    31. Ctrl + . Quick fix
    32. Selecionando algum texto aperte Ctrl + F Ele ira navegar por textos iguais
    33. Selecionando algum texto aperte Ctrl + H Será possível modificar o texto
